# Research Sources

These are the primary sources the research agent should monitor and prioritize when gathering information. The agent should also search broadly beyond these, but these represent Sondra's trusted information ecosystem.

## YouTube Channels (AI Agents & Technical)

- **Cole Medin** — AI agent masterclass, multi-agent workflows, code tutorials. Widely considered one of the best technical educators for building AI agents.
- **David Shapiro** — Philosophy, design, and architecture of autonomous agents and swarms. ACE framework. The "why" behind agentic systems.
- **AI Jason (Jason Zhou)** — Agentic AI tutorials, practical step-by-step guides.
- **Brandon (aiwithbrandon)** — Business use cases of AI agents, real-world problem solving.
- **Alejandro AO** — RAG chains and AI-driven applications, crash-course style.
- **Sam Witteveen** — Google Developer Expert, LangChain agents, self-reasoning agents, deep technical understanding.

## GitHub Repositories

### Agentic Frameworks
- **LangChain / LangGraph** — Industry standard for orchestration. LangGraph for stateful multi-actor agents.
- **LlamaIndex** — Evolved from RAG to full agentic document processing and agentic retrieval.
- **n8n** — Visual node-based workflow automation for AI-driven agents (160k+ stars).
- **Microsoft AutoGen** — Multi-agent conversational systems.
- **CrewAI** — Role-playing autonomous AI agents, built on LangChain.

### Advanced RAG
- **NirDiamant/RAG_Techniques** (27k+ stars) — Multi-HyDE, Graph RAG, query enhancement, with Jupyter notebook tutorials.
